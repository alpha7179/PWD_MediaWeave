## 2. 디자인 원칙 (Design Principles)

### 2.1. 핵심 가치

1. **명확성 (Clarity)**
   - 복잡한 커리큘럼 정보의 직관적 시각화
   - 불필요한 장식 배제, 정보 전달 집중

2. **일관성 (Consistency)**
   - 모든 페이지에서 동일한 패턴 사용
   - 학습 곡선 최소화

3. **접근성 (Accessibility)**
   - WCAG 2.1 AA 기준 준수
   - 색상 대비 비율 4.5:1 이상
   - 키보드 네비게이션 지원

4. **반응성 (Responsiveness)**
   - 모바일 우선 설계 (Mobile-First)
   - 데스크톱, 태블릿, 모바일 최적화

## 11. 접근성 가이드라인 (Accessibility Guidelines)

### 11.1. 색상 대비 (Color Contrast)

#### WCAG 2.1 AA 기준 준수

**일반 텍스트 (16px 미만)**
- 최소 대비 비율: 4.5:1
- 예시: `#111827` (text-primary) on `#FFFFFF` (bg-primary) = 16.1:1 ✓

**큰 텍스트 (18px 이상 또는 14px Bold)**
- 최소 대비 비율: 3:1
- 예시: `#6B7280` (text-secondary) on `#FFFFFF` = 4.6:1 ✓

**UI 컴포넌트 및 그래픽 요소**
- 최소 대비 비율: 3:1
- 예시: 버튼 테두리, 입력 필드 테두리

#### 색상 대비 검증 테이블

| 전경색 | 배경색 | 대비 비율 | WCAG AA | WCAG AAA |
|--------|--------|----------|---------|----------|
| text-primary (#111827) | bg-primary (#FFFFFF) | 16.1:1 | ✓ | ✓ |
| text-secondary (#6B7280) | bg-primary (#FFFFFF) | 4.6:1 | ✓ | ✗ |
| primary-600 (#4F46E5) | bg-primary (#FFFFFF) | 7.3:1 | ✓ | ✓ |
| error-500 (#EF4444) | bg-primary (#FFFFFF) | 4.5:1 | ✓ | ✗ |
| success-700 (#047857) | success-50 (#ECFDF5) | 7.8:1 | ✓ | ✓ |

#### 색상만으로 정보 전달 금지

```html
<!-- ❌ 잘못된 예시: 색상만으로 상태 표시 -->
<span style="color: red;">오류</span>

<!-- ✓ 올바른 예시: 아이콘 + 텍스트 + 색상 -->
<span class="text-error-500">
  <svg class="icon-sm">...</svg>
  오류: 필수 입력 항목
</span>
```

### 11.2. 키보드 네비게이션 (Keyboard Navigation)

#### Tab 순서 (Tab Order)

- 모든 인터랙티브 요소는 `Tab` 키로 접근 가능
- 논리적 순서로 포커스 이동 (좌→우, 상→하)
- `tabindex="0"`: 자연스러운 Tab 순서에 포함
- `tabindex="-1"`: Tab으로 접근 불가, JavaScript로만 포커스 가능
- `tabindex="1+"`: 사용 금지 (자연스러운 순서 방해)

#### Focus 상태 스타일

```css
/* 모든 포커스 가능 요소 */
*:focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
  border-radius: var(--border-radius-sm);
}

/* 버튼 포커스 */
.btn:focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
}

/* 입력 필드 포커스 */
.input:focus-visible {
  outline: none;
  border-color: var(--primary-500);
  box-shadow: 0 0 0 3px rgba(99, 102, 241, 0.1);
}

/* 링크 포커스 */
a:focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
  border-radius: var(--border-radius-sm);
}
```

#### 키보드 단축키

| 키 | 동작 |
|----|------|
| `Tab` | 다음 요소로 포커스 이동 |
| `Shift + Tab` | 이전 요소로 포커스 이동 |
| `Enter` | 버튼/링크 활성화 |
| `Space` | 버튼 활성화, 체크박스 토글 |
| `Esc` | 모달/드롭다운 닫기 |
| `Arrow Keys` | 드롭다운/메뉴 내 이동 |

#### Skip Link 구현
